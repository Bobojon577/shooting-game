Action script This script will be an abstract class and so script is going to be an abstract class so we’re never gonna actually instantiate of action we are going to use it as a base class for all our other actions and it is going to take a state controller variable called controller.
State script Both our functions have a state controller parameter and this is because we’re continuously passing through a reference to whichever mono behavior is calling these functions of our state assets. You might wonder why we don’t just create a state controller variable for a class and use that and the reason is that they are scriptable object and will need to be referred to by multiple agents and multiple shooters in the scene. If we allow variable on it, that will become a value for any other object references. Added the create asset menu attribute and pass to the menu name "CreateAssetMenu (menuName = "PluggableAI/State")" than a public array of actions called actions and will set those in the inspector into actions we want to loop over. The action will be loop over by DoAction state and our "actions [i].Act (controller);" and we’re gonna pass in our controller.
StateController This script will contain public CurrentState of AI, enemyState to define the AI target, eyes of AI(picture above of AI shoots prefab), if target is dead the AI will acticate the remainState. There also hiden variables such as navMeshAgent to make AI able to navigate on ground, Shooting script, list of waypoint to search for Enemy shooters and chasing target compopnent.
Now in previous blog 4.0 we have a shooter that will patrol until it sees the player and then chase until the player dies. The next step is to add a second action, which is attacking, so we don't want our enemy to be able to fire a bullet every frame. Our attack action is going to need a way to check a timer and since each tank is going to need to keep track of its own time to actions. We're going to locate this in our state controller which is the amount of behavior attached to our shooter.
